Kickoffs - ReactJS - Agenda Manager

 

Build a react application that should manage agenda. It should helps to add agenda title and topics and view those agenda details.

 

This application have buttons[Click To View Agenda, Click To Add Agenda] to switch between add agenda form(for adding agenda details) and view agenda(for displaying the details).

Form Fields And its Criteria
Title: Validators [required]
Description:  Validators [required] 
Topic:  Validators [required] 

 

  • Check for empty inputs and show error message if empty value is given in inputs - the validation messages are provided in the template

  • The form have Add Topic and Submit Agenda button.

  • Add Topic button should be used for add the agenda topics as much as need to add. After all the needed topics added and title and description inputs are given, Submit Agenda button should be used for adding the agenda details.

  • Add button should be disabled if the input field(Topic) is empty. Submit Agenda button should be disabled if the Title or Description is empty or no topics added yet.

  • An error message should shown(present in the template - No Topics Added) if no topics added yet

  • On changing the view, the added details should shown.

 

Instructions to install and run front-end

  • If you want to install project(ReactJS) dependecies, run sh install.sh in terminal. Do not install any additional dependencies, it may affect your scoring.
  • If you want to run your application, run sh run.sh in terminal (you can view your application on port 8000 in browser)
  • To run test cases, run sh test.sh in terminal

 

Note:

  • Follow the instructions given in the code to complete the challenge.
